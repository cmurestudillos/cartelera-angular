# Peliculas - Angular
Aplicacion web de peliculas realizada con Angular utilizando la API [themoviedb](https://www.themoviedb.org).

## Requisitos
- Node.js 20+
- [pnpm](https://pnpm.io/) (unico gestor de paquetes soportado por el proyecto)

## Instalar dependencias
Ejecutar en terminal: `pnpm install`.

## Iniciar proyecto
Ejecutar en terminal el comando `pnpm start` (equivalente a `ng serve`) e introducir en el navegador `http://localhost:4200/`.

## Scripts disponibles
- `pnpm start` — servidor de desarrollo
- `pnpm run build` — build de produccion (`dist/browser`)
- `pnpm test` — tests con Vitest
- `pnpm run lint` — ESLint (flat config)

## Ayuda
Para mas informacion, ejecutar el comando `pnpm exec ng help` en el terminal o ir a [Angular CLI README](https://github.com/angular/angular-cli/blob/master/README.md).

Este proyecto se genero originalmente con Angular CLI 8.3.23 y ha sido migrado a Angular 22 + pnpm.
